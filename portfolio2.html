POS for a coffee shop
A point-of-sale system created in .NET framework together with C# and phpmyAdmin as its database. This is our capstone project for the semester, it is supposed to be implemented in kiosks, kitchen screens and cash registers. We developed 3 systems seperately for each purpose. An admin system, A cashier system and the ordering/kiosk system. The admin system displays the sales and inventory statistics fetched from the database. Next is the cashier system that works like any other cash register where you punch orders and total the price that sends the info to the database; which will then be displayed to the kitchen screen for cooks to tally what items are to be served. All this is connected by a unified database that adds, deducts and sorts itself based on the formula written by us four students.
